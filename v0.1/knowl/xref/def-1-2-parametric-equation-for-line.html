Definition 1.2.4. The Parametric Equation For a Line.

Consider a \(3\)-vector \(\mathbf{d} = (d_1,d_2,d_3)\) and a point \(\mathbf{x}_0 = (x_0,y_0,z_0)\) in three-dimensional space. Then the line through a point \(\mathbf{x}_0\) with direction vector \(\mathbf{d}\) is equal to the set of all points of the form
\begin{equation*} \mathbf{x}_0 + t \mathbf{d} = (x_0 + td_1, y_0 + td_2, z_0 + td_3)\text{.} \end{equation*}
We can also write the line symmetrically as the set of points \(\mathbf{x} = (x,y,z)\) such that
\begin{equation*} \frac{x - x_0}{d_1} = \frac{y - y_0}{d_2} = \frac{z - z_0}{d_3}\text{.} \end{equation*}
All three values in this equation are equal to the scalar \(t\) which occurs in the first equations.
