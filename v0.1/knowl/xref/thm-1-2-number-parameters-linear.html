Theorem 1.4.12.

Suppose a system of \(m\) equations in \(n\) variables is consistent (see Definition 1.3.3), and the rank of the augmented matrix is \(r\text{.}\) Then
  1. The set of solutions is specified in exactly \(n-r\) parameters.
  2. If \(r \lt n\text{,}\) the system has infinitely many solutions.
  3. If \(r = n\text{,}\) the system has a unique solution.
