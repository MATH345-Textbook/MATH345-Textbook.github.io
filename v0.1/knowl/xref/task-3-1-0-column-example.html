Task 3.1.5.b.
\begin{equation*} B = \begin{bmatrix} 0 \amp \pi \amp 200\\ 0 \amp 0.03 \amp 4 \\ 0 \amp 19 \amp -3584\end{bmatrix} \end{equation*}
Solution.
If we expand the determinant of \(B\) along it’s first column, we find that
\begin{equation*} \det(B) = 0\cdot\text{cof}(A)_{11} + 0\cdot\text{cof}(A)_{21} + 0\cdot\text{cof}(A)_{31} = 0\text{.} \end{equation*}
