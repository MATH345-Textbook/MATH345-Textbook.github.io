Definition 1.5.3. Basic Solutions To Homogeneous Linear Equations.

The Gaussian elimination algorithm produces, for each free variable, a solution to the homogeneous equation \(A \mathbf{x} = \mathbf{0}\text{,}\) only by setting that free variable equal to one, and all other free variables equal to \(0\text{.}\) These solutions are called basic solutions, one for every parameter. Every solution can be written uniquely as a linear combination of the basic solutions.
