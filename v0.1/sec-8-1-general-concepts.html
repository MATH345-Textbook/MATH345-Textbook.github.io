    Section 7.1 Vector-Valued Functions

    We now begin our study of the calculus of multivariable functions, i.e. functions of more than one scalar variable. You’ve likely studied the calculus of functions of a single variable, but many real-world applications of calculus involve the study of functions depending on multiple varying quantities.

    Definition 7.1.1.

    A vector-valued function from \(\R^n\) to \(\R^m\text{,}\) is a function from \(\R^n\) to \(\R^m\text{,}\) typically denoted \(\mathbf{f}: \R^n \to \R^m\text{.}\) A scalar-valued function is a function \(f: \R^n \to \R\text{,}\) i.e. a vector-valued function from \(\R^n\) to \(\R^m\) with \(m = 1\text{.}\) Any vector-valued function \(\mathbf{f}: \R^n \to \R^m\) can be expressed in terms of \(m\) component functions, or coordinate functions, the scalar-valued functions \(f_1, \dots, f_m: \R^n \to \R\) such that
    \begin{equation*} \mathbf{f}(\mathbf{x}) = ( f_1(\mathbf{x}), \dots, f_m(\mathbf{x}) )\text{.} \end{equation*}
    More generally, vector-valued and scalar-valued functions may only be defined on a subset of vectors in \(\R^n\text{,}\) rather than defined everywhere, i.e. their domain may be a proper subset of \(\R^n\text{.}\)
    In other words, a scalar-valued function takes in an input and outputs scalars, and a vector-valued function takes in an input and outputs vectors (recall Definition 1.1.1).

    Activity 7.1.1.

    If \(\mathbf{x}\) is a point in \(\R^n\text{,}\) recall the length of \(\mathbf{x}\text{,}\) denoted \(\| \mathbf{x} \|\text{,}\) from Definition 1.1.5. The length of a vector defines a scalar-valued function from \(\R^n\) to \(\R\text{,}\) i.e. the function which takes in a vector \(\mathbf{x}\) as input and gives the length \(\| \mathbf{x} \|\) as output. Show that this function has the property that two different inputs can have the same output. Recall from Definition 4.4.7 that we say this function is not injective.
    Solution.
    There are many different choices one could make here. For instance, the vector \((3,4)\) and the vector \((5,0)\) both have the same length. Other examples are the vectors \((1,2)\) and \((-1,2)\text{.}\)

    Activity 7.1.2.

    Let \(A\) be an \(m \times n\) matrix. The linear transformation (recall Definition 2.4.1) \(T: \R^n \to \R^m\) defined by \(T(\mathbf{x}) = T_A(\mathbf{x}) = A\mathbf{x}\) is a vector-valued function.

    (a)

    If \(\mathbf{a}_1, \dots, \mathbf{a}_m\) are the rows of \(A\text{,}\) and \(T_1, \dots, T_m\) are the component functions of \(T\text{,}\) then for each \(j\text{,}\) express the function \(T_j\) in terms of the row \(\mathbf{a}_j\text{.}\)
    Solution.
    Recalling the definition of matrix vector multiplication from Definition 2.2.2, we can write
    \begin{equation*} T_j(\mathbf{x}) = \mathbf{a}_{j1} x_1 + \cdots + \mathbf{a}_{jn} x_n = \mathbf{a}_j \cdot \mathbf{x}\text{.} \end{equation*}

    (b)

    Consider the matrix
    \begin{equation*} A = \begin{pmatrix} 1 & 2 & -1 \\ 0 & 3 & 4 \end{pmatrix}\text{,} \end{equation*}
    which defines a linear transformation \(T_A: \R^3 \to \R^2\text{.}\) Compute its component functions for a general input \(\mathbf{x}\text{,}\) and its output for the specific input \(\mathbf{x} = (3, -1, 2)\text{.}\)
    Solution.
    The component functions are:
    \begin{equation*} T_1(\mathbf{x}) = (1,2,-1) \cdot \mathbf{x} = x_1 + 2 x_2 - x_3 \end{equation*}
    and
    \begin{equation*} T_2(\mathbf{x}) = (0,3,4) \cdot \mathbf{x} = 3x_2 + 4x_3\text{.} \end{equation*}
    For an input \(\mathbf{x} = (3, -1, 2)\text{,}\) we compute that
    \begin{equation*} T(\mathbf{x}) = \begin{pmatrix} 1 \amp 2 \amp -1 \\ 0 \amp 3 \amp 4 \end{pmatrix} \begin{pmatrix} 3 \\ -1 \\2 \end{pmatrix} = \begin{pmatrix} -1 \\ 5 \end{pmatrix}\text{.} \end{equation*}
