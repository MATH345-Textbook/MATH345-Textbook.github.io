    Section 6.3 The Singular Value Decomposition

    Recall Figure 6.2.6, which gives a visualization of the action of a matrix \(A\) in terms of it’s orthogonal diagonalization. The Singular Value Decomposition (the SVD) is a powerful generalization of the diagonalization guaranteed by Theorem 6.2.8, that works for all matrices, of any size, rather than just symmetric square matrices. Rather than writing a square matrix \(A\) as \(QDQ^T\text{,}\) where \(Q\) is an orthogonal basis and \(D\) is a diagonal matrix, the singular value decomposition writes an \(m \times n\) matrix \(A\) of any size as \(P \Sigma Q^T\text{,}\) where \(P\) is an orthogonal \(m \times m\) matrix, \(Q\) is an orthogonal \(n \times n\) matrix, and \(\Sigma\) is a diagonal \(m \times n\) matrix with non-negative entries (recall the definition of a diagonal matrix of any size given in Definition 2.1.9).
    Recall that the diagonalization of a symmetric \(n \times n\) matrix \(A\) is equivalent to finding an orthonormal basis \(\{\mathbf{x}_1,\ldots,\mathbf{x}_n\}\) of eigenvectors such that \(A\mathbf{x}_i = \lambda_i\mathbf{x}_i\text{.}\) Recalling Remark 6.1.12, it is not possible to diagonalize a general square matrix. The Singular Value Decomposition of a \(m \times n\) matrix \(A\) instead is equivalent to finding a pair of orthonormal bases \(\{ \mathbf{p}_1, \dots, \mathbf{p}_m \}\) and \(\{ \mathbf{q}_1, \dots, \mathbf{q}_m \}\) so that for \(1 \leq i \leq \min(m,n)\text{,}\) \(A \mathbf{q}_i = \sigma_i \mathbf{p}_i\) for each \(i\text{,}\) for scalars \(\sigma_1 \geq \cdots \geq \sigma_n \geq 0\) which are called the singular values of \(A\text{,}\) and if \(m \leq n\text{,}\) \(A \mathbf{q}_i = 0\) for \(i \gt m\text{.}\)
    Geometrically, the SVD gives another point of view about how a matrix transforms space. Recall that the absolute values of the eigenvalues of a square symmetric matrix measure the amount of “stretching” of space along the directions spanned by the eigenvetors of the matrix. In the singular value decomposition of a matrix \(A\text{,}\) the singular value \(\sigma_1\) measures the maximum amount of stretching of the matrix \(A\text{,}\) which occurs along the vectors \(\mathbf{q}_1\) and \(\mathbf{p}_1\text{.}\)
    More precisely, we can write
    \begin{equation*} \| Ax \|^2 = (Ax) \cdot (Ax) = x^T (A^T A) x\text{.} \end{equation*}
    Since \(A^T A\) is symmetric, it can be orthogonally diagonalized, and the maximum value of \(\| Ax \|^2\) (subject to the normalization \(\| x \| = 1\)) is the largest eigenvalue of \(A^T A\text{,}\) which equals to \(\sigma_1^2\text{.}\) Moreover, this maximization occurs when \(\mathbf{x} = \mathbf{q}_1\text{.}\) The other singular vectors gives the other directions of stretching.
    This property (among others) makes the singular value decomposition one of the most important tools in applied linear algebra, with applications ranging from data compression and image processing to statistical analysis.
    An \(m \times n\) matrix \(A\) maps the unit sphere (the set of all unit vectors) in \(\R^n\) to an ellipsoid in \(\mathbb{R}^m\) (i.e., a sphere which has been stretched and compressed along different principal axes). The singular value decomposition identifies the principal axes of this ellipsoid and the corresponding scaling factors. Figure 6.3.1 gives an example of how this may be visualized when \(A\) is a \(2 \times 3\) matrix.

    Lemma 6.3.2. Properties of \(A^TA\) and \(AA^T\).

    Let \(A\) be an \(m\times n\) matrix. Then:
    1. The eigenvalues of \(A^TA\) and \(AA^T\) are real and non-negative.
    2. \(A^TA\) and \(AA^T\) have the same set of positive eigenvalues.

    Proof.

    1. The Matrices \(A^T A\) and \(A A^T\) are symmetric, so by Theorem 6.2.8 their eigenvalues are real. If \(A^T A \mathbf{q} = \lambda \mathbf{q}\) for \(\mathbf{q} \neq \mathbf{0}\text{,}\) we calculate that
    \begin{align*} \|A\mathbf{q}\|^2 \amp = (A\mathbf{q})^T(A\mathbf{q}) = \mathbf{q}^TA^TA\mathbf{q} = \mathbf{q}^T\lambda\mathbf{q} = \lambda\|\mathbf{q}\|^2 \end{align*}
    Since \(\|A\mathbf{q}\|^2 \geq 0\) and \(\|\mathbf{q}\|^2 > 0\text{,}\) we conclude that \(\lambda \geq 0\text{.}\)
    2. Let \(\lambda\) be a positive eigenvalue of \(A^TA\) with eigenvector \(\mathbf{q} \neq \mathbf{0}\text{.}\) Then \(A^TA\mathbf{q} = \lambda\mathbf{q}\text{.}\) Let \(\mathbf{p} = A\mathbf{q}\text{.}\) Note that \(\mathbf{p} \neq \mathbf{0}\) since \(\lambda > 0\text{.}\) We have:
    \begin{align*} AA^T\mathbf{p} \amp = AA^T(A\mathbf{q}) = A(A^TA\mathbf{q}) = A(\lambda\mathbf{q}) = \lambda\mathbf{p}. \end{align*}
    So \(\lambda\) is also an eigenvalue of \(AA^T\text{.}\) The argument works in reverse by replacing \(A\) with \(A^T\text{,}\) proving that \(A^TA\) and \(AA^T\) have the same positive eigenvalues.

    Theorem 6.3.3. Computing the Singular Value Decomposition.

    Given an \(m \times n\) matrix \(A\text{,}\) we can obtain a SVD of \(A\) of the form
    \begin{equation*} A = P\Sigma Q^T \end{equation*}
    as follows:
    1. Let \(\lambda_1, \lambda_2, \ldots, \lambda_n\) be the eigenvalues of \(A^TA\) with corresponding (orthonormal) eigenvectors \(\mathbf{q}_1, \mathbf{q}_2, \ldots, \mathbf{q}_n\text{.}\) Reorder the \(\mathbf{q}_i\) (if necessary) to ensure that the nonzero eigenvalues are \(\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_r > 0\) and \(\lambda_i = 0\) if \(i > r\text{.}\)
    2. Let \(r\) be the rank of the matrix \(A\text{.}\)
    3. Define \(Q = [\mathbf{q}_1~\mathbf{q}_2~\cdots~\mathbf{q}_n]\text{.}\)
    4. Define
      \begin{equation*} \mathbf{p}_i = \frac{1}{\|A\mathbf{q}_i\|}A\mathbf{q}_i \end{equation*}
      for \(i = 1, 2, \ldots, r\text{.}\) Then \(\{\mathbf{p}_1, \mathbf{p}_2, \ldots, \mathbf{p}_r\}\) is orthonormal in \(\mathbb{R}^m\text{,}\) so (using Gram-Schmidt or otherwise) extend the set to an orthonormal basis \(\{\mathbf{p}_1, \ldots, \mathbf{p}_r, \ldots, \mathbf{p}_m\}\) in \(\mathbb{R}^m\text{.}\)
    5. Define \(P = [\mathbf{p}_1~\cdots~\mathbf{p}_r~\cdots~\mathbf{p}_m]\text{.}\)
    6. The singular values for \(A\) are \(\sigma_1, \sigma_2, \ldots, \sigma_n\) where \(\sigma_i = \sqrt{\lambda_i}\) for each \(i\text{.}\) The nonzero singular values are \(\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_r > 0\text{,}\) and we define
      \begin{equation*} \Sigma = \begin{bmatrix} \diag (\sigma_1, \ldots, \sigma_r) \amp 0 \\ 0 \amp 0 \end{bmatrix}_{m \times n}\text{.} \end{equation*}

    Proof.

    We start by justifying that \(r\) is the rank of the matrix \(A\text{,}\) and that the vectors \(\{ A \mathbf{q}_i : i \leq r \}\) are orthogonal. For \(i \neq j\text{,}\) since \(\{\mathbf{q}_1, \mathbf{q}_2, \ldots, \mathbf{q}_n\}\) is an orthonormal set,
    \begin{align*} A\mathbf{q}_i \cdot A\mathbf{q}_j \amp = (A\mathbf{q}_i)^T(A\mathbf{q}_j)\\ \amp = \mathbf{q}_i^T(A^TA)\mathbf{q}_j\\ \amp = \mathbf{q}_i^T(\lambda_j\mathbf{q}_j)\\ \amp = \lambda_j(\mathbf{q}_i \cdot \mathbf{q}_j)\\ \amp = \lambda_j \cdot 0 = 0\text{.} \end{align*}
    The vectors \(A\mathbf{q}_i\) for \(i > r\) are zero, since \(\lambda_i = 0\) implies
    \begin{equation*} \|A\mathbf{q}_i\|^2 = \lambda_i\|\mathbf{q}_i\|^2 = 0\text{.} \end{equation*}
    Let \(\mathbf{x} \in \mathbb{R}^n\) be any vector. Since \(\{\mathbf{q}_1, \ldots, \mathbf{q}_n\}\) is a basis of \(\mathbb{R}^n\text{,}\) we can write \(\mathbf{x} = \sum_{i=1}^n t_i\mathbf{q}_i\) for some scalars \(t_i\text{.}\) Then:
    \begin{align*} A\mathbf{x} \amp = A\left(\sum_{i=1}^n t_i\mathbf{q}_i\right)\\ \amp = \sum_{i=1}^n t_iA\mathbf{q}_i\\ \amp = \sum_{i=1}^r t_iA\mathbf{q}_i + \sum_{i=r+1}^n t_iA\mathbf{q}_i\\ \amp = \sum_{i=1}^r t_iA\mathbf{q}_i \end{align*}
    Thus \(\im(A) \subseteq \spans \{A\mathbf{q}_1, A\mathbf{q}_2, \ldots, A\mathbf{q}_r\}\text{.}\) Since \(\im(A)\) is a subspace, and \(A \mathbf{q}_i\) are in \(\im(A)\text{,}\) we have equality. Since \(\{A\mathbf{q}_1, A\mathbf{q}_2, \ldots, A\mathbf{q}_r\}\) is an orthogonal set, it is linearly independent, and thus a basis of \(\im(A)\text{.}\) So \(\rank (A) = r\text{,}\) since \(\rank (A) = \dimens(\im(A)) = r\text{.}\)
    Now we prove that we have obtained a singular value decomposition of \(A\text{.}\) We compute that
    \begin{align*} P\Sigma_A \amp = [\mathbf{p}_1 \cdots \mathbf{p}_r \cdots \mathbf{p}_m] \begin{bmatrix} \sigma_1 \amp \cdots \amp 0 \amp 0 \amp \cdots \amp 0 \\ \vdots \amp \ddots \amp \vdots \amp \vdots \amp \amp \vdots \\ 0 \amp \cdots \amp \sigma_r \amp 0 \amp \cdots \amp 0 \\ 0 \amp \cdots \amp 0 \amp 0 \amp \cdots \amp 0 \\ \vdots \amp \amp \vdots \amp \vdots \amp \amp \vdots \\ 0 \amp \cdots \amp 0 \amp 0 \amp \cdots \amp 0 \end{bmatrix}\\ \amp = \begin{bmatrix} \sigma_1\mathbf{p}_1 \amp \cdots \amp \sigma_r\mathbf{p}_r \amp \mathbf{0} \amp \cdots \amp \mathbf{0} \end{bmatrix} \end{align*}
    Recall that \(\sigma_i\mathbf{p}_i = \sqrt{\lambda_i}\mathbf{p}_i = \|A\mathbf{q}_i\|\mathbf{p}_i = A\mathbf{q}_i\) for \(i = 1,2,\ldots,r\text{.}\) Therefore
    \begin{align*} P\Sigma_A \amp = [A\mathbf{q}_1 \cdots A\mathbf{q}_r~\mathbf{0} \cdots \mathbf{0}]\\ \amp = [A\mathbf{q}_1 \cdots A\mathbf{q}_r~A\mathbf{q}_{r+1} \cdots A\mathbf{q}_n]\\ \amp = AQ \end{align*}
    Since \(Q\) is orthogonal, \(Q^{-1} = Q^T\text{,}\) and therefore multiplying the identity \(P \Sigma_A = A Q\) by \(Q^T\) on the right, we conclude that \(A = P\Sigma_A Q^T\text{.}\)

    Remark 6.3.4.

    The Singular Value Decomposition of a matrix is not unique. Any matrix factorization \(P \Sigma Q^T\) with the same properties (\(P\) and \(Q\) are orthogonal and \(\Sigma\) is diagonal with non-negative, non-increasing diagonal entries) is also called an SVD. The same results hold for any SVD, e.g., that the number of positive singular values is equal to the rank of \(A\text{.}\)

    Activity 6.3.1.

    Find a singular value decomposition of the matrix
    \begin{equation*} A = \begin{bmatrix} 1 \amp -1 \\ -2 \amp 2 \\ 2 \amp -2 \end{bmatrix}\text{.} \end{equation*}
    Use this decomposition to compute the eigenvalues and eigenvectors of \(A^T A\text{.}\)
    Solution.
    We compute that
    \begin{align*} A^TA \amp = \begin{bmatrix} 1 \amp -2 \amp 2 \\ -1 \amp 2 \amp -2 \end{bmatrix} \begin{bmatrix} 1 \amp -1 \\ -2 \amp 2 \\ 2 \amp -2 \end{bmatrix}\\ \amp = \begin{bmatrix} 9 \amp -9 \\ -9 \amp 9 \end{bmatrix} \end{align*}
    Now we diagonalize \(A^T A\text{.}\) We compute the characteristic polynomial of the matrix, i.e.
    \begin{align*} \det(A^TA - \lambda I) \amp = \det\begin{bmatrix} 9-\lambda \amp -9 \\ -9 \amp 9-\lambda \end{bmatrix}\\ \amp = (9-\lambda)^2 - 81\\ \amp = \lambda(\lambda - 18)\text{.} \end{align*}
    Thus the eigenvalues of \(A^T A\) are \(\lambda_1 = 18\) and \(\lambda_2 = 0\text{.}\) For \(\lambda_1 = 18\text{,}\) we solve the equation \((A^TA - 18I)\mathbf{q}_1 = \mathbf{0}\text{,}\) i.e.
    \begin{equation*} \begin{bmatrix} -9 \amp -9 \\ -9 \amp -9 \end{bmatrix}\begin{bmatrix} x \\ y \end{bmatrix} = \begin{bmatrix} 0 \\ 0 \end{bmatrix}\text{,} \end{equation*}
    which has a basic solution
    \begin{equation*} \begin{bmatrix} -1 \\ 1 \end{bmatrix}\text{.} \end{equation*}
    Since we need a unit vector, we take
    \begin{equation*} \mathbf{q}_1 = \frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ -1 \end{bmatrix} = \begin{bmatrix} (1/2) \sqrt{2} \\ (-1/2) \sqrt{2} \end{bmatrix}\text{.} \end{equation*}
    For \(\lambda_2 = 0\text{,}\) we solve \(A^TA\mathbf{q}_2 = \mathbf{0}\text{,}\) and obtain that
    \begin{equation*} \begin{bmatrix} 9 \amp -9 \\ -9 \amp 9 \end{bmatrix}\begin{bmatrix} x \\ y \end{bmatrix} = \begin{bmatrix} 0 \\ 0 \end{bmatrix}\text{,} \end{equation*}
    which has a basic solution
    \begin{equation*} \begin{bmatrix} 1 \\ 1 \end{bmatrix}\text{.} \end{equation*}
    Renormalizing to obtain a unit vector, we obtain that
    \begin{equation*} \mathbf{q}_2 = \frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ 1 \end{bmatrix} = \begin{bmatrix} (1/2) \sqrt{2} \\ (1/2) \sqrt{2} \end{bmatrix}\text{.} \end{equation*}
    The vectors \(\{ \mathbf{q}_1, \mathbf{q}_2 \}\) are the columns of the orthogonal matrix \(Q\text{.}\)
    Next, we compute the quantities \(\{ \sigma_i \}\) and vectors \(\{ \mathbf{p}_i \}\text{.}\) We have
    \begin{equation*} \sigma_1 = \sqrt{\lambda_1} = \sqrt{18} = 3 \sqrt{2} \end{equation*}
    and
    \begin{equation*} \sigma_2 = \sqrt{\lambda_2} = 0\text{.} \end{equation*}
    So \(r = 1\text{.}\) We have
    \begin{align*} A\mathbf{q}_1 \amp = \begin{bmatrix} 1 \amp -1 \\ -2 \amp 2 \\ 2 \amp -2 \end{bmatrix}\frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ -1 \end{bmatrix} = \frac{1}{\sqrt{2}}\begin{bmatrix} 2 \\ -4 \\ 4 \end{bmatrix} \end{align*}
    Normalizing we obtain the first column of \(P\text{,}\) i.e.
    \begin{equation*} \mathbf{p}_1 = \frac{A\mathbf{q}_1}{\|A\mathbf{q}_1\|} = \frac{1}{3}\begin{bmatrix} 1 \\ -2 \\ 2 \end{bmatrix} \end{equation*}
    The other vectors \(\mathbf{p}_2\) and \(\mathbf{p}_3\) can be obtained by taking any pair of vectors orthonormal to \(\mathbf{p}_1\text{.}\) Applying Gram-Schmidt to
    \begin{equation*} \mathbf{v}_1 = \begin{bmatrix} 1 \\ -2 \\ 2 \end{bmatrix} \quad \mathbf{v}_2 = \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix} \quad \mathbf{v}_3 = \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix}\text{,} \end{equation*}
    to obtain an orthogonal basis \(\mathbf{w}_1, \mathbf{w}_2, \mathbf{w}_3\text{,}\) we set \(\mathbf{w}_1 = \mathbf{v}_1\text{,}\) \(\mathbf{w}_2\) to a scalar multiple of
    \begin{align*} \mathbf{v}_2 - \amp \frac{\mathbf{w}_1 \cdot \mathbf{v}_2}{\| \mathbf{w}_1 \|^2} \mathbf{w}_1\\ \amp = \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix} - \frac{1}{9} \begin{bmatrix} 1 \\ -2 \\ 2 \end{bmatrix}\\ \amp = \begin{bmatrix} 8/9 \\ 2/9 \\ -2/9 \end{bmatrix}\text{,} \end{align*}
    i.e. setting
    \begin{equation*} \mathbf{w}_2 = \begin{bmatrix} 8 \\ 2 \\ -2 \end{bmatrix}\text{,} \end{equation*}
    and then defining \(\mathbf{w}_3\) to be a scalar multiple of
    \begin{align*} \mathbf{v}_3 - \amp \frac{\mathbf{w}_1 \cdot \mathbf{v}_3}{\| \mathbf{w}_1 \|^2} \mathbf{w}_1 - \frac{\mathbf{w}_2 \cdot \mathbf{v}_3}{\| \mathbf{w}_2 \|^2} \mathbf{w}_2\\ \amp = \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} - \frac{-2}{9} \begin{bmatrix} 1 \\ -2 \\ 2 \end{bmatrix} - \frac{2}{72} \begin{bmatrix} 8 \\ 2 \\ -2 \end{bmatrix}\\ \amp = \begin{bmatrix} 0 \\ 1/2 \\ 1/2 \end{bmatrix}\text{,} \end{align*}
    i.e. setting
    \begin{equation*} \mathbf{w}_3 = \begin{bmatrix} 0 \\ 1 \\ 1 \end{bmatrix}\text{.} \end{equation*}
    Normalizing these vectors, we obtain the vectors
    \begin{equation*} \mathbf{p}_1 = \frac{\mathbf{w}_1}{\| \mathbf{w}_1 \|} = \frac{\mathbf{w}_1}{\sqrt{9}} = \begin{bmatrix} 1/3 \\ -2/3 \\ 2/3 \end{bmatrix}, \end{equation*}
    \begin{equation*} \mathbf{p}_2 = \frac{\mathbf{w}_2}{\| \mathbf{w}_2 \|} = \frac{\mathbf{w}_2}{\sqrt{72}} = \begin{bmatrix} (2/3) \sqrt{2} \\ (1/6) \sqrt{2} \\ (-1/6) \sqrt{2} \end{bmatrix} \end{equation*}
    and
    \begin{equation*} \mathbf{p}_3 = \frac{\mathbf{w}_3}{\| \mathbf{w}_3 \|} = \frac{\mathbf{w}_3}{\sqrt{2}} = \begin{bmatrix} 0 \\ (1/2) \sqrt{2} \\ (1/2) \sqrt{2} \end{bmatrix}\text{.} \end{equation*}
    We thus define
    \begin{equation*} Q = \begin{bmatrix} \mathbf{q}_1 \amp \mathbf{q}_2 \end{bmatrix} = \sqrt{2} \begin{bmatrix} 1/2 \amp 1/2 \\ -1/2 \amp 1/2 \end{bmatrix}\text{,} \end{equation*}
    we define
    \begin{equation*} \Sigma = \begin{bmatrix} 3\sqrt{2} \amp 0 \\ 0 \amp 0 \\ 0 \amp 0 \end{bmatrix}\text{,} \end{equation*}
    and
    \begin{equation*} P = \begin{bmatrix} \mathbf{p}_1 \amp \mathbf{p}_2 \amp \mathbf{p}_3 \end{bmatrix} = \begin{bmatrix} 1/3 \amp (2/3) \sqrt{2} \amp 0 \\ -2/3 \amp (1/6) \sqrt{2} \amp (1/2) \sqrt{2} \\ 2/3 \amp (-1/6) \sqrt{2} \amp (1/2) \sqrt{2} \end{bmatrix} \end{equation*}
    It turns out that any singular value decomposition contains a great deal of information about an \(m \times n\) matrix \(A\) and the subspaces associated with \(A\text{.}\)

    Definition 6.3.5. Fundamental Subspaces.

    The four fundamental subspaces corresponding to a \(m\times n\) matrix \(A\) are:
    • The row space \(\row A = \spans \{\mathbf{x} \mid \mathbf{x} \text{ is a row of } A\}\text{.}\)
    • The column space \(\col A = \spans \{\mathbf{x} \mid \mathbf{x} \text{ is a column of } A\}\text{.}\)
    • The null space \(\nulls A = \{\mathbf{x} \in \mathbb{R}^n \mid A\mathbf{x} = \mathbf{0}\}\text{.}\)
    • The null space of \(A^T\text{,}\) i.e. \(\nulls A^T = \{\mathbf{x} \in \mathbb{R}^m \mid A^T\mathbf{x} = \mathbf{0}\}\text{.}\)

    Theorem 6.3.6. Relations Between Singular Value Decomposition and Fundamental Subspaces.

    Let \(A\) be an \(m \times n\) matrix, and let \(A = P \Sigma Q^T\) be a singular value decomposition for \(A\text{.}\) Let
    \begin{equation*} \Sigma = \begin{bmatrix} D \amp 0 \\ 0 \amp 0 \end{bmatrix}\text{,} \end{equation*}
    where \(D = \diag (d_1,\dots,d_r)\text{,}\) with \(d_1,\dots,d_r > 0\text{.}\) Let \(\mathbf{p}_1,\dots,\mathbf{p}_m\) and \(\mathbf{q}_1,\dots,\mathbf{q}_n\) be the columns of \(P\) and \(Q\) respectively. Then
    • \(\{\mathbf{p}_1, \ldots, \mathbf{p}_r\}\) is an orthonormal basis of \(\col(A)\text{.}\)
    • \(\{\mathbf{p}_{r+1}, \ldots, \mathbf{p}_m\}\) is an orthonormal basis of \(\nulls(A^T)\text{.}\)
    • \(\{\mathbf{q}_{r+1}, \ldots, \mathbf{q}_n\}\) is an orthonormal basis of \(\nulls(A)\text{.}\)
    • \(\{\mathbf{q}_1, \ldots, \mathbf{q}_r\}\) is an orthonormal basis of \(\row(A)\text{.}\)
