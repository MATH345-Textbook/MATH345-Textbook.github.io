    Section 4.4 Other Vector Spaces

    The results we have proved about subspaces of \(\R^n\) apply more generally to finite-dimensional vector spaces, which are sets of objects which we can add and multiply by scalars in ways entirely analogous to row and column vectors. We will define vector spaces in the following section in Definition 4.5.1. In this section we give some more vector spaces that show up in practical application of linear algebra here.

    Subsection Vector Spaces of Polynomials

    Definition 4.4.1. The Vector Space of Polynomials.

    A polynomial is a function expressible as
    \begin{equation*} p(x) = a_0 + a_1 x + a_2 x^2 + \cdots + a_d x^d \end{equation*}
    where \(a_0,\dots,a_d\) are real numbers, and \(d\) is a non-negative integer. If \(a_d \neq 0\) we say that \(p\) has degree \(d\text{.}\) Let \(\mathbf{P}_d\) be the set of all polynomials of degree at most \(d\text{,}\) together with the zero polynomial \(p(x) = 0\text{,}\) denoted \(\mathbf{0}\text{.}\)

    Definition 4.4.2.

    A polynomial is a function expressible as
    \begin{equation*} p(x) = a_0 + a_1 x + a_2 x^2 + \cdots + a_n x^n \end{equation*}
    where \(a_0,\dots,a_n\) are real numbers, and \(n\) is a non-negative integer. If \(a_n \neq 0\) we say that \(p\) has degree \(n\text{.}\) We let \(\mathbf{P}\) denote the set of all polynomials, and \(\mathbf{P}_n\) the set of all polynomials with degree at most \(n\text{.}\)
    One can add two polynomials to get a new polynomial, and multiply a polynomial by a scalar to get a new polynomial, so the sets \(\mathbf{P}_n\) behave analogously to column vectors, and so we can apply all the terminology we have developed for column vectors in the past chapter to polynomials.

    Activity 4.4.1.

    Consider the set \(U\) of all polynomials in \(\mathbf{P}_4\) that have \(-2\) as a root, i.e. the set
    \begin{equation*} U = \{ p \in \mathbf{P}_4 : p(-2) = 0 \}\text{.} \end{equation*}
    Show that \(U\) is a subspace of \(\mathbf{P}_4\text{.}\)
    Solution.
    Recall the definition of subspaces given in Definition 4.1.1. The set \(U\) is a subspace of \(\mathbf{P}_4\) precisely when \(U\) contains the zero polynomial \(\mathbf{0}\) (the polynomial such that \(\mathbf{0}(x) = 0\) for all \(x \in \R\)), and is closed under addition and scalar multiplication:
    Contains the zero polynomial
    \(\mathbf{0}(-2) = 0\text{,}\) so \(\mathbf{0} \in U\text{.}\)
    Closure Under Addition
    If \(p\) and \(q\) are in \(U\text{,}\) then \(p(-2) = 0\) and \(q(-2) = 0\text{,}\) and
    \begin{equation*} (p + q)(-2) = p(-2) + q(-2) = 0 + 0 = 0\text{,} \end{equation*}
    so \(p + q \in U\text{.}\)
    Closure under Scalar Multiplication
    If \(p \in U\) and \(a\) is a scalar, then \(p(-2) = 0\) and so
    \begin{equation*} (ap)(-2) = a p(-2) = a 0 = 0\text{,} \end{equation*}
    so \(ap \in U\text{.}\)
    Since all three conditions are satified, \(U\) is a subspace of \(\mathbf{P}_4\text{.}\)

    Activity 4.4.2.

    Consider the polynomials
    \begin{equation*} p_1(x) = 2 - x + 3x^2 \quad\text{and}\quad p_2(x) = 1 + 2x - x^2 \end{equation*}
    in \(\mathbf{P}_2\text{.}\) Determine whether \(p_3(x) = 5 + x + x^2\) belongs to the span of \(p_1\) and \(p_2\text{.}\)
    Solution.
    We must determine whether there are scalars \(a\) and \(b\) such that
    \begin{equation*} p_3 = ap_1 + bp_2\text{.} \end{equation*}
    This can only be possible if the coefficients of the polynomials on each side of the equation are equal to one another. Expanding, we have
    \begin{align*} 5 + 1 x + 1 x^2 \amp = a(2 - x + 3x^2) + b(1 + 2x - x^2)\\ \amp = (2a + b) + (-a + 2b) x + (3a - b) x^2 \end{align*}
    so
    \begin{align*} 5 \amp = 2a + b\\ 1 \amp = -a + 2b\\ 1 \amp = 3a - b \end{align*}
    Solving this system using one of the various methods we have considered in this class shows this system is inconsistent, with no solution, and so \(p_3 \not \in \spans \{ p_1, p_2 \}\text{.}\)

    Activity 4.4.3.

    Consider the ordered basis \(B = (x+1, 3)\) for \(\mathbf{P}_1\text{.}\) Find \(C_B(\mathbf{v})\) for \(\mathbf{v} = 2x - 4\text{.}\)
    Solution.
    We must find scalars \(a\) and \(b\) such that
    \begin{equation*} 2x - 4 = a(x+1) + b(3)\text{.} \end{equation*}
    Expanding gives
    \begin{equation*} -4 + 2x = (a + 3b) + a x \end{equation*}
    which gives \(a + 3b = -4\) and \(a = 2\text{.}\) Solving gives \(a = 2\) and \(b = -2\text{,}\) and so
    \begin{equation*} C_B(\mathbf{v}) = \begin{bmatrix} 2 \\ - 2 \end{bmatrix}\text{.} \end{equation*}

    Subsection Vector Spaces of Matrices

    Definition 4.4.3.

    For each pair of positive integers \(n\) and \(m\text{,}\) we let \(\mathbf{M}_{nm}\) denote the set of all \(n \times m\) matrices with real entries. This set is also sometimes denoted by \(\mathbf{M}_{n \times m}\) or \(\mathbf{R}^{n \times m}\text{.}\)
    Just as for polynomials, we can add two matrices together, and multiply matrices by scalars, so the terminology we have introduced in this chapter can also be applied to \(\mathbf{M}_{nm}\text{.}\)

    Activity 4.4.4.

    Suppose that \(A\) is a \(n \times n\) matrix such that \(A^k = 0\) but \(A^{k-1} \neq 0\text{.}\) Show that the set \(B = \{ I, A, A^2, \dots, A^{k-1} \}\) is linearly independent in \(\mathbf{M}_{nm}\text{.}\)
    Solution.
    Suppose that
    \begin{equation*} t_0 I + t_1 A + t_2 A^2 + \cdots + t_{k-1} A^{k-1} = 0\text{.} \end{equation*}
    If we multiply both sides of the equation by \(A^{k-1}\text{,}\) we find that (since \(A^i = 0\) for all \(i \geq k\)),
    \begin{equation*} 0 = A^{k-1}(t_0 I + t_1 A + t_2 A^2 + \cdots + t_{k-1} A^{k-1}) = t_0 A^{k-1}\text{.} \end{equation*}
    Thus \(t_0 = 0\text{.}\) Multiplying by \(A^{k-2}\text{,}\) we find that
    \begin{equation*} 0 = t_1 A^{k-1}\text{,} \end{equation*}
    and thus that \(t_1\text{.}\) Repeatedly applying this process by multiplying by \(A^{k-j}\text{,}\) we find that \(t_{j-1} = 0\) for all \(j\text{.}\) Therefore \(B\) is linearly independent.

    Activity 4.4.5.

    Let
    \begin{equation*} A = \begin{bmatrix} 1 \amp 1 \\ 0 \amp 0 \end{bmatrix} \end{equation*}
    be a fixed matrix in \(\mathbf{M}_{22}\text{.}\) Define the set
    \begin{equation*} U = \{ X \in \mathbf{M}_{22} : AX = XA \}\text{.} \end{equation*}
    (a)
    Show that \(U\) is a subspace of \(\mathbf{M}_{22}\text{.}\)
    Solution.
    The set \(U\) is a subspace of \(\mathbf{P}_4\) precisely when \(U\) contains the zero matrix and is closed under addition and scalar multiplication:
    Contains the zero matrix
    \(A \mathbf{0} = 0\) and \(\mathbf{0} A = 0\text{,}\) so \(A \mathbf{0} = \mathbf{0} A\text{,}\) so \(\mathbf{0} \in \mathbf{M}_{22}\text{.}\)
    Closure Under Addition
    If \(X_1,X_2 \in U\text{,}\) then to verify \(X_1 + X_2 \in U\) we must check that \(A(X_1 + X_2) = (X_1 + X_2) A\text{.}\) But \(AX_1 = X_1A\) and \(AX_2 = X_2A\text{,}\) so
    \begin{align*} A(X_1 + X_2) \amp = AX_1 + AX_2\\ \amp = X_1A + X_2A\\ \amp = (X_1 + X_2) A\text{.} \end{align*}
    Closure under Scalar Multiplication
    If \(X \in U\) and \(c\) is a scalar, then to verify \(c X \in U\) we must check that \(A(cX) = (cX)A\text{.}\) But
    \begin{equation*} A(cX) = c(AX) = c(XA) = (cX)A\text{.} \end{equation*}
    Since all three conditions are satified, \(U\) is a subspace of \(\mathbf{M}_{22}\text{.}\)
    (b)
    Find a spanning set for \(U\text{.}\)
    Solution.
    Let
    \begin{equation*} X = \begin{bmatrix} x \amp y \\ z \amp w \end{bmatrix}\text{.} \end{equation*}
    Then
    \begin{equation*} AX = \begin{bmatrix} 1 \amp 1 \\ 0 \amp 0 \end{bmatrix} \begin{bmatrix} x \amp y \\ z \amp w \end{bmatrix} = \begin{bmatrix} x + z \amp y + w \\ 0 \amp 0 \end{bmatrix} \end{equation*}
    and
    \begin{equation*} XA = \begin{bmatrix} x \amp y \\ z \amp w \end{bmatrix} \begin{bmatrix} 1 \amp 1 \\ 0 \amp 0 \end{bmatrix} = \begin{bmatrix} x \amp x \\ z \amp z \end{bmatrix}\text{.} \end{equation*}
    Thus comparing entries of these matrices, \(AX = XA\) if and only if \(x + z = x\text{,}\) \(y + w = x\text{,}\) and \(0 = z\text{.}\) Solving these equations gives \(z = 0\) and \(x = y + w\text{,}\) and so an arbitrary entry of \(U\) can be written as
    \begin{equation*} \begin{bmatrix} y + w \amp y \\ 0 \amp w \end{bmatrix} = y \begin{bmatrix} 1 \amp 1 \\ 0 \amp 0 \end{bmatrix} + w \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} \text{.} \end{equation*}
    Thus
    \begin{equation*} U = \spans \left\{ \begin{bmatrix} 1 \amp 1 \\ 0 \amp 0 \end{bmatrix}, \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} \right\} \end{equation*}
    (c)
    Show that \(\dimens(U) = 2\) and find a basis for \(U\text{.}\)
    Solution.
    In the previous part of this activity, we verified that
    \begin{equation*} U = \spans \left\{ \begin{bmatrix} 1 \amp 1 \\ 0 \amp 0 \end{bmatrix}, \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} \right\} \end{equation*}
    We claim that
    \begin{equation*} \left\{ \begin{bmatrix} 1 \amp 1 \\ 0 \\ 0 \end{bmatrix}, \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} \right\} \end{equation*}
    is also a basis, and to do this it suffices to show that
    \begin{equation*} a \begin{bmatrix} 1 \amp 1 \\ 0 \amp 0 \end{bmatrix} + b \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} = 0 \end{equation*}
    has only the trivial solution \(a = 0\) and \(b = 0\text{.}\) But we can simplify this equation to read
    \begin{equation*} \begin{bmatrix} a + b \amp a \\ 0 \amp b \end{bmatrix} \end{equation*}
    and this matrix can only be equal to zero if \(a = 0\) and \(b = 0\text{.}\)
    Since the basis we have found has two elements, this implies that \(U\) is two dimensional.

    Subsection Linear Transformations in Abstract Vector Spaces

    Recall that a function \(f: A \to B\) between two sets \(A\) and \(B\) is a rule that assigns to each \(a \in A\) a unique \(f(a) \in B\text{.}\) Two functions \(f\) and \(g\) are equal if \(f(a) = g(a)\) for all \(a \in A\text{.}\)

    Definition 4.4.4. Linear Transformations.

    A function \(T: V \to W\) between two vector spaces \(V\) and \(W\) is a linear transformation if it satifies the following two conditions:
    Preserves Addition
    \(T(\mathbf{v}_1 + \mathbf{v}_2) = T(\mathbf{v}_1) + T(\mathbf{v}_2)\) for all \(\mathbf{v}_1,\mathbf{v}_2 \in V\text{.}\)
    Preserves Scalar Multiplication
    \(T(c \mathbf{v}) = c T(\mathbf{v})\) for all \(v \in V\) and all scalars \(c\text{.}\)
    A linear transformation \(T: V \to V\) is called a linear operator.

    Note 4.4.5.

    The first condition of a linear transformation states that adding two vectors in \(V\text{,}\) and then applying a linear transformation in \(T\text{,}\) is the same as first applying \(T\text{,}\) and then adding the resulting vectors in \(W\text{.}\) Note that, despite using the addition symbol \(+\) on both sides of the equation, these operations denote different addition operations: the operation on the left performed in \(V\text{,}\) and the operation on the right performed in \(W\text{.}\)

    Definition 4.4.6. Kernels and Images of Linear Transformations.

    The kernel of a linear transformation \(T: V \to W\) is
    \begin{equation*} \kers(T) = \{ \mathbf{v} \in V: T(\mathbf{v}) = \mathbf{0} \}\text{.} \end{equation*}
    The image of \(T\) is
    \begin{equation*} \im(T) = \{ T(\mathbf{v}) : \mathbf{v} \in V \}\text{.} \end{equation*}
    The kernel of \(T\) is also called the nullspace of \(T\) because it consists of all solutions to the equation \(T(\mathbf{v}) = \mathbf{0}\text{.}\) The image of \(T\) is also called the range of \(T\text{,}\) and consists of all vectors \(\mathbf{w} \in W\) for which the equation \(\mathbf{w} = T(\mathbf{v})\) is solvable in \(V\text{.}\)

    Definition 4.4.7.

    \(X\)\(Y\)\(f: X \to Y\)injectiveone-to-one\(f\)\(f(x_1) = f(x_2)\)\(x_1,x_2 \in X\)\(x_1 = x_2\text{.}\)\(f\)surjectiveonto\(Y\)\(f(x)\)\(x \in X\text{.}\)
    For linear transformations \(T: V \to W\text{,}\) the transformation is injective if and only if \(\kers(T) = \{ 0 \}\) (since the equation \(T(x_1) = T(x_2)\) is equivalent to \(T(x_1 - x_2) = 0\text{,}\) i.e. to \(x_1 - x_2 \in \text{Ker}(T)\)). The transformation is surjective if and only if \(\im(T) = W\text{.}\)

    Activity 4.4.6.

    Let \(D: \mathbf{P}_n \to \mathbf{P}_{n-1}\) be the differentiation map \(D(p) = p'\text{.}\)
    (a)
    Show that \(D\) is a linear transformation.
    Solution.
    First, note that \(D\) is a well defined function from \(\mathbf{P}_n\) to \(\mathbf{P}_{n-1}\text{,}\) because the derivative of a polynomial with degree at most \(n\) is a polynomial with degree at most \(n-1\text{.}\)
    That \(D\) preserves addition follows because of the sum rule \((f + g)' = f' + g'\) for the derivative, i.e. because
    \begin{equation*} D(p + q) = (p + q)' = p' + q' = D(p) + D(q)\text{.} \end{equation*}
    That \(D\) preserves scalar multiplication follows because of the rule \((cf)' = cf'\) for scalars \(c\text{,}\) so that
    \begin{equation*} D(cp) = (cp)' = cp' = cD(p)\text{.} \end{equation*}
    Thus \(D\) satisfies both properties necessary and sufficient for \(D\) to be a linear transformation.
    (b)
    Compute \(\kers(D)\text{.}\)
    Solution.
    It suffices to find all polynomials \(p\) with degree at most \(n\) such that \(p'(x) = 0\) for all \(x \in \R\text{.}\) The only functions whose derivative is equal to zero everywhere are constants, so \(\kers(D)\) consists precisely of the set of constant polynomials.
