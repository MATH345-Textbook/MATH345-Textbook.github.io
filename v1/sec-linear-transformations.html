    Section 2.4 Linear Transformations

    What makes a transformation a matrix transformation?

    Definition 2.4.1. Linear Transformations.

    A transformation \(T:\R^n\to\R^m\) is a linear transformation if
    1. \(T(\mathbf{x} + \mathbf{y}) = T(\mathbf{x}) + T(\mathbf{y})\) for every \(\mathbf{x}, \mathbf{y} \in \R^n\text{.}\)
    2. \(T(c\mathbf{x})=cT(\mathbf{x})\) for every \(\mathbf{x} \in \R^n\) and \(c \in \R\text{.}\)

    Activity 2.4.1. Verifying Linear Transformations.

    Identifying \(\R\) with \(\R^1\text{,}\) show that \(T(x)=2x\) is a linear transformation, while \(T(x)=x^2\) is not.
    Solution.
    For \(T(x)=2x\text{:}\)
    \begin{equation*} T(x+y) = 2(x+y) = 2x + 2y = T(x) + T(y) \end{equation*}
    and
    \begin{equation*} T(cx) = 2(cx) = c(2x) = cT(x) \end{equation*}
    Therefore \(T(x)=2x\) is a linear transformation.
    For \(T(x)=x^2\text{:}\)
    \begin{equation*} T(x+y) = (x+y)^2 = x^2 + 2xy + y^2 \end{equation*}
    and
    \begin{equation*} T(x) + T(y) = x^2 + y^2 \end{equation*}
    but it is not always true (and in fact, is almost never true) that
    \begin{equation*} x^2 + 2xy + y^2 = x^2 + y^2\text{.} \end{equation*}
    So it is not always true that \(T(x + y) = T(x) + T(y)\text{,}\) and so \(T(x)=x^2\) is not a linear transformation.

    Activity 2.4.2. Properties of Linear Transformations.

    \(T\)

    (a)

    Find \(T(\mathbf{0})\text{.}\)
    Solution.
    Since \(\mathbf{0} = 0 \mathbf{0}\text{,}\)
    \begin{equation*} T(\mathbf{0}) = T(0\mathbf{0}) = 0T(\mathbf{0})\text{,} \end{equation*}
    so \(T(\mathbf{0}) = \mathbf{0}\text{.}\)

    (b)

    \(T(-\mathbf{x})\text{.}\)
    Solution.
    \begin{equation*} T(-\mathbf{x}) = T((-1)\mathbf{x}) = (-1)T(\mathbf{x}) = -T(\mathbf{x})\text{.} \end{equation*}
    Recall the notion of a linear combination from Definition 1.5.1, i.e. that a vector \(\mathbf{y}\in\R^n\) is a linear combination of vectors \(\mathbf{x}_1, \mathbf{x}_2,\ldots \mathbf{x}_k\) if there are scalars \(a_1, a_2, \ldots, a_k\) such that
    \begin{equation*} \mathbf{y}=a_1\mathbf{x}_1+a_2\mathbf{x}_2+\cdots a_k\mathbf{x}_k. \end{equation*}

    Theorem 2.4.2.

    If \(T:\R^n\to\R^m\) is a linear transformation, then
    \begin{equation*} T(a_1\mathbf{x}_1+a_2\mathbf{x}_2+\cdots a_k\mathbf{x}_k)= a_1T(\mathbf{x}_1) + a_2T(\mathbf{x}_2) + \cdots + a_kT(\mathbf{x}_k) \end{equation*}

    Activity 2.4.3. Finding Images Under Linear Transformations.

    Suppose \(T:\R^2\to\R^3\) is a linear transformation with
    \begin{equation*} T((1,2)) = (-1,0,3) \end{equation*}
    and
    \begin{equation*} T((-1,0)) = (4,1,2)\text{.} \end{equation*}
    Find
    \begin{equation*} T((3,4))\text{.} \end{equation*}
    Solution.
    We plan to use Theorem 2.4.2. To do this, we need to write the vector \((3,4)\) as a linear combination of \((1,2)\) and \((-1,0)\text{.}\)
    Let \(a,b\) be scalars such that:
    \begin{equation*} \begin{bmatrix} 3 \\ 4 \end{bmatrix} = a\begin{bmatrix} 1 \\ 2 \end{bmatrix} + b\begin{bmatrix} -1 \\ 0 \end{bmatrix} \end{equation*}
    In order for this equation to hold, the scalars \(a\) and \(b\) must satisfy the system of linear equations:
    \begin{gather*} a - b = 3\\ 2a = 4 \end{gather*}
    From the second equation: \(a = 2\)
    Substituting into the first equation: \(2 - b = 3\text{,}\) so \(b = -1\text{.}\)
    Therefore,
    \begin{equation*} (3,4) = 2 (1,2) - (-1,0)\text{.} \end{equation*}
    By the linearity of \(T\) and Theorem 2.4.2,
    \begin{align*} T((3,4)) \amp = 2 T((1,2)) - T((-1,0))\\ &= 2 (-1,0,3) - (4,1,2)\\ &= \begin{bmatrix} -2 \\ 0 \\ 6 \end{bmatrix} - \begin{bmatrix} 4 \\ 1 \\ 2 \end{bmatrix}\\ &= \begin{bmatrix} -6 \\ -1 \\ 4 \end{bmatrix} \end{align*}

    Theorem 2.4.3.

    Let \(T:\R^n\to\R^m\) be a transformation.
    1. \(T\) is a linear transformation if and only if it is a matrix transformation.
    2. In this case, \(T=T_A\) is the matrix transformation induced by the unique matrix \(A\text{,}\) given in terms of its columns by
    \begin{equation*} A=\begin{bmatrix} T(\mathbf{e}_1) & T(\mathbf{e}_2) & \cdots & T(\mathbf{e}_n) \end{bmatrix}, \end{equation*}
    where \(\mathbf{e}_1,\mathbf{e}_2, \ldots \mathbf{e}_n\) are the standard basis vectors of \(\R^n\text{.}\)

    Proof.

    First, we show that if \(T\) is a matrix transformation, then it is linear. If \(T = T_A\) for a matrix \(A\text{,}\) then the distributive properties from Theorem 2.2.3 show that:
    \begin{equation*} T(\mathbf{x} + \mathbf{y}) = A(\mathbf{x} + \mathbf{y}) = A\mathbf{x} + A\mathbf{y} = T(\mathbf{x}) + T(\mathbf{y}) \end{equation*}
    and
    \begin{equation*} T(c\mathbf{x}) = A(c\mathbf{x}) = cA\mathbf{x} = cT(\mathbf{x}) \end{equation*}
    By Definition 2.4.1, this shows \(T_A\) is linear.
    Now we show that if \(T\) is any linear transformation, then it is a matrix transformation. Define a matrix
    \begin{equation*} A = \begin{bmatrix} T(\mathbf{e}_1) & T(\mathbf{e}_2) & \cdots & T(\mathbf{e}_n) \end{bmatrix}\text{.} \end{equation*}
    For any
    \begin{equation*} \mathbf{x} = \begin{bmatrix} x_1 \\ x_2 \\ \vdots \\ x_n \end{bmatrix}\text{,} \end{equation*}
    we can write:
    \begin{equation*} \mathbf{x} = x_1\mathbf{e}_1 + x_2\mathbf{e}_2 + \cdots + x_n\mathbf{e}_n \end{equation*}
    By linearity,
    \begin{align*} T(\mathbf{x}) &= T(x_1\mathbf{e}_1 + x_2\mathbf{e}_2 + \cdots + x_n\mathbf{e}_n)\\ &= x_1T(\mathbf{e}_1) + x_2T(\mathbf{e}_2) + \cdots + x_nT(\mathbf{e}_n)\\ &= A\mathbf{x} \end{align*}
    Therefore \(T = T_A\) is a matrix transformation.
    To show that a linear transformation is a matrix transformation for a unique matrix \(A\text{,}\) suppose \(T = T_B\) for some other matrix \(B\text{.}\) Then it would be true that \(T_A \mathbf{e}_j = T_B \mathbf{e}_j\) for each standard basis vector \(\mathbf{e}_j\text{.}\) But (recall Fact 2.2.10) \(T_A \mathbf{e}_j = A\mathbf{e}_j\) is the \(j\)th column of \(A\text{,}\) and \(T_B \mathbf{e}_j = B\mathbf{e}_j\) is the \(j\)th column of \(B\text{.}\) Therefore the columns of \(A\) and \(B\) are identical, so \(A = B\text{.}\)

    Corollary 2.4.4.

    If \(T:\R^k\to\R^n\text{,}\) and \(S:\R^n\to\R^m\) are linear transformations, then \(S\circ T\) is a linear transformation.

    Activity 2.4.4. Finding Matrix Representation.

    Define \(T:\R^2\to\R^3\) by
    \begin{equation*} T\left(\begin{bmatrix} x_1 \\ x_2 \end{bmatrix}\right)=\begin{bmatrix} x_1 +x_2 \\ x_1 \\ 3x_2 \end{bmatrix} \end{equation*}

    (a)

    Show \(T\) is a linear transformation.
    Solution.
    Let’s verify both properties of linear transformations:
    \begin{align*} T(\mathbf{x} + \mathbf{y}) &= \begin{bmatrix} (x_1+y_1)+(x_2+y_2) \\ x_1+y_1 \\ 3(x_2+y_2) \end{bmatrix}\\ &= \begin{bmatrix} x_1+x_2 \\ x_1 \\ 3x_2 \end{bmatrix} + \begin{bmatrix} y_1+y_2 \\ y_1 \\ 3y_2 \end{bmatrix}\\ &= T(\mathbf{x}) + T(\mathbf{y}) \end{align*}
    and
    \begin{align*} T(c\mathbf{x}) &= \begin{bmatrix} cx_1+cx_2 \\ cx_1 \\ 3cx_2 \end{bmatrix}\\ &= c\begin{bmatrix} x_1+x_2 \\ x_1 \\ 3x_2 \end{bmatrix}\\ &= cT(\mathbf{x}) \end{align*}

    (b)

    Find \(A\text{,}\) where \(T = T_A\text{.}\)
    Solution.
    To find \(A\text{,}\) we need \(T(\mathbf{e}_1)\) and \(T(\mathbf{e}_2)\text{:}\)
    \begin{align*} T(\mathbf{e}_1) &= T\begin{bmatrix} 1 \\ 0 \end{bmatrix} = \begin{bmatrix} 1 \\ 1 \\ 0 \end{bmatrix}\\ T(\mathbf{e}_2) &= T\begin{bmatrix} 0 \\ 1 \end{bmatrix} = \begin{bmatrix} 1 \\ 0 \\ 3 \end{bmatrix} \end{align*}
    Therefore:
    \begin{equation*} A = \begin{bmatrix} T\mathbf{e}_1 \amp T\mathbf{e}_2 \end{bmatrix} = \begin{bmatrix} 1 & 1 \\ 1 & 0 \\ 0 & 3 \end{bmatrix}\text{.} \end{equation*}

    (c)

    Is there an \(\mathbf{x}\) in \(\R^2\) with \(T(\mathbf{x})=\begin{bmatrix} 4 \\ -1 \\ 6\end{bmatrix}\text{?}\)
    Solution.
    We need to solve the equation
    \begin{equation*} A\mathbf{x} = \begin{bmatrix} 4 \\ -1 \\ 6 \end{bmatrix}\text{.} \end{equation*}
    This gives the system
    \begin{equation*} \begin{array}{cccccc} x_1 \amp + \amp x_2 \amp = \amp 4\\ x_1 \amp \amp \amp = \amp -1\\ \amp \amp 3x_2 \amp = \amp 6 \end{array} \end{equation*}
    From second equation: \(x_1 = -1\)
    From third equation: \(x_2 = 2\)
    If \(x_1 = -1\) and \(x_2 = 2\text{,}\) the first equation reads \(-1 + 2 = 4\text{,}\) which is never true. So the system has no solution, and so there is no such \(\mathbf{x}\text{.}\)
