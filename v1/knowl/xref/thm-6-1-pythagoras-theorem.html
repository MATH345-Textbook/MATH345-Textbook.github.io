Theorem 5.1.7. Pythagoras’ Theorem.

If \(\{\mathbf{f}_1, \mathbf{f}_2, \dots, \mathbf{f}_k \}\) is an orthogonal set in \(\R^n\text{,}\) then
\begin{equation*} \|\mathbf{f}_1 + \mathbf{f}_2 + \cdots + \mathbf{f}_k\|^2 = \|\mathbf{f}_1\|^2 + \|\mathbf{f}_2\|^2 + \cdots + \|\mathbf{f}_k\|^2. \end{equation*}
