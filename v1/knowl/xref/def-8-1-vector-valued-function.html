Definition 7.1.1.

A vector-valued function from \(\R^n\) to \(\R^m\text{,}\) is a function from \(\R^n\) to \(\R^m\text{,}\) typically denoted \(\mathbf{f}: \R^n \to \R^m\text{.}\) A scalar-valued function is a function \(f: \R^n \to \R\text{,}\) i.e. a vector-valued function from \(\R^n\) to \(\R^m\) with \(m = 1\text{.}\) Any vector-valued function \(\mathbf{f}: \R^n \to \R^m\) can be expressed in terms of \(m\) component functions, or coordinate functions, the scalar-valued functions \(f_1, \dots, f_m: \R^n \to \R\) such that
\begin{equation*} \mathbf{f}(\mathbf{x}) = ( f_1(\mathbf{x}), \dots, f_m(\mathbf{x}) )\text{.} \end{equation*}
More generally, vector-valued and scalar-valued functions may only be defined on a subset of vectors in \(\R^n\text{,}\) rather than defined everywhere, i.e. their domain may be a proper subset of \(\R^n\text{.}\)
