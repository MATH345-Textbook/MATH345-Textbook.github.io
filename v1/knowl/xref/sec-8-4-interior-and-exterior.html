Definition 7.4.9. Interior and Exterior Points of Sets.

Let \(S\) be a subset of \(\R^n\text{.}\) Then a point \(\mathbf{x}_0 \in S\) is an interior point if there is a \(\delta\) ball centered at \(\mathbf{x}_0\) completely contained in \(S\text{.}\) A point \(\mathbf{x}_0 \in \R^n\) is a boundary point if it is not an interior point, i.e. if every \(\delta\) disk centered around \(\mathbf{x}_0\) contains both points inside \(S\text{,}\) and also points outside of \(S\text{.}\) The set of all boundary points of a set \(S\) is called the boundary of the set \(S\text{.}\)
