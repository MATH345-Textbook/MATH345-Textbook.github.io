Definition 1.2.1. The Normal Vector Equation For Planes.

Consider a vector \(\mathbf{n} \in \R^3\text{.}\) Then for any point \(\mathbf{x}_0\) in three-dimensional space, we can consider the plane through \(\mathbf{x}_0\text{,}\) with normal vector \(\mathbf{n}\text{,}\) to be the set of all points \(\mathbf{x}\) such that the vector \(\mathbf{x} - \mathbf{x}_0\) is perpendicular to \(\mathbf{n}\text{,}\) i.e.
\begin{equation*} \mathbf{n} \cdot (\mathbf{x} - \mathbf{x}_0) = 0\text{.} \end{equation*}
This vector equation can be expanded into an equation involving scalars. If we write \(\mathbf{x} = (x,y,z)\text{,}\) \(\mathbf{x}_0 = (x_0,y_0,z_0)\text{,}\) and \(\mathbf{n} = (n_1,n_2,n_3)\text{,}\) then the equation describing the plane can be writen as
\begin{equation*} n_1(x - x_0) + n_2(y - y_0) + n_3(z - z_0) = 0\text{.} \end{equation*}
