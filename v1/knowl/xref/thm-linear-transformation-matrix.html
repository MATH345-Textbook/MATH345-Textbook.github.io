Theorem 2.4.3.

Let \(T:\R^n\to\R^m\) be a transformation.
  1. \(T\) is a linear transformation if and only if it is a matrix transformation.
  2. In this case, \(T=T_A\) is the matrix transformation induced by the unique matrix \(A\text{,}\) given in terms of its columns by
\begin{equation*} A=\begin{bmatrix} T(\mathbf{e}_1) & T(\mathbf{e}_2) & \cdots & T(\mathbf{e}_n) \end{bmatrix}, \end{equation*}
where \(\mathbf{e}_1,\mathbf{e}_2, \ldots \mathbf{e}_n\) are the standard basis vectors of \(\R^n\text{.}\)

Proof.

First, we show that if \(T\) is a matrix transformation, then it is linear. If \(T = T_A\) for a matrix \(A\text{,}\) then the distributive properties from Theorem 2.2.3 show that:
\begin{equation*} T(\mathbf{x} + \mathbf{y}) = A(\mathbf{x} + \mathbf{y}) = A\mathbf{x} + A\mathbf{y} = T(\mathbf{x}) + T(\mathbf{y}) \end{equation*}
and
\begin{equation*} T(c\mathbf{x}) = A(c\mathbf{x}) = cA\mathbf{x} = cT(\mathbf{x}) \end{equation*}
By Definition 2.4.1, this shows \(T_A\) is linear.
Now we show that if \(T\) is any linear transformation, then it is a matrix transformation. Define a matrix
\begin{equation*} A = \begin{bmatrix} T(\mathbf{e}_1) & T(\mathbf{e}_2) & \cdots & T(\mathbf{e}_n) \end{bmatrix}\text{.} \end{equation*}
For any
\begin{equation*} \mathbf{x} = \begin{bmatrix} x_1 \\ x_2 \\ \vdots \\ x_n \end{bmatrix}\text{,} \end{equation*}
we can write:
\begin{equation*} \mathbf{x} = x_1\mathbf{e}_1 + x_2\mathbf{e}_2 + \cdots + x_n\mathbf{e}_n \end{equation*}
By linearity,
\begin{align*} T(\mathbf{x}) &= T(x_1\mathbf{e}_1 + x_2\mathbf{e}_2 + \cdots + x_n\mathbf{e}_n)\\ &= x_1T(\mathbf{e}_1) + x_2T(\mathbf{e}_2) + \cdots + x_nT(\mathbf{e}_n)\\ &= A\mathbf{x} \end{align*}
Therefore \(T = T_A\) is a matrix transformation.
To show that a linear transformation is a matrix transformation for a unique matrix \(A\text{,}\) suppose \(T = T_B\) for some other matrix \(B\text{.}\) Then it would be true that \(T_A \mathbf{e}_j = T_B \mathbf{e}_j\) for each standard basis vector \(\mathbf{e}_j\text{.}\) But (recall Fact 2.2.10) \(T_A \mathbf{e}_j = A\mathbf{e}_j\) is the \(j\)th column of \(A\text{,}\) and \(T_B \mathbf{e}_j = B\mathbf{e}_j\) is the \(j\)th column of \(B\text{.}\) Therefore the columns of \(A\) and \(B\) are identical, so \(A = B\text{.}\)
