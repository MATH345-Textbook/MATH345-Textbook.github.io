    Section 2.2 Matrix-Vector Multiplication

    Subsection Vectors as Matrices

    Recall the notion of row and column vectors from Definition 1.5.1. Row vectors and column vectors can be identified with the vectors occuring in Definition 1.1.1, i.e. an ordered list of scalars. The only difference is that in a row vector the ordered list of numbers is arranged horizontally, and for a column vector the ordered list is arranged vertically. The distinguish between these types of vectors, we will say “a row vector in \(\R^n\)” and “a column vector in \(\R^n\)”.

    Activity 2.2.1.

    Consider the vector
    \begin{equation*} \mathbf{v}=\begin{bmatrix} -3 \\ \sqrt{2} \\ 0\end{bmatrix}\text{.} \end{equation*}
    What kind of vector is it?
    Solution.
    \(\mathbf{v}\) is a column vector in \(\R^3\text{.}\)

    Note 2.2.1.

    An \(m\times n\) matrix \(A\) can be thought of a row of \(n\) column vectors of length \(m\text{,}\) i.e. we can write \(A=\begin{bmatrix}\mathbf{a}_1 \amp \mathbf{a}_2 \amp\cdots \amp \mathbf{a}_n\end{bmatrix}\text{,}\) where \(\mathbf{a}_i\) is a column vector in \(\mathbf{R}^m\text{,}\) the i-th column of \(A\text{.}\)

    Subsection Matrix-Vector Product

    We start with an example.

    Activity 2.2.2. Linear Systems as Linear Combinations.

    Consider the linear system
    \begin{equation*} \begin{array}{ccccccccc} x \amp + \amp 2y \amp + \amp 3z \amp \amp = \amp \amp 6\\ 2x \amp - \amp 3y \amp + \amp 2z \amp \amp = \amp \amp 14\\ 3x \amp + \amp y \amp - \amp z \amp \amp = \amp \amp -2 \end{array} \end{equation*}
    Rewrite this system of linear equations as a single equation involving linear combinations of vectors.
    Solution.
    \begin{equation*} x \begin{bmatrix} 1 \\ 2 \\ 3 \end{bmatrix} + y \begin{bmatrix} 2 \\ -3 \\ 1 \end{bmatrix} + z \begin{bmatrix} 3 \\ 2 \\ -1 \end{bmatrix} = \begin{bmatrix} 6 \\ 14 \\ -2 \end{bmatrix} \end{equation*}

    Definition 2.2.2. Matrix-Vector Products.

    Let \(A=\begin{bmatrix}\mathbf{a}_1 \amp \mathbf{a}_2 \amp\cdots \amp \mathbf{a}_n\end{bmatrix}\) be an \(m\times n\) matrix, written in terms of its columns \(\mathbf{a}_1, \mathbf{a}_2, \ldots, \mathbf{a}_n\text{.}\) If
    \begin{equation*} \mathbf{x}=\begin{bmatrix} x_1 \\ x_2 \\ \vdots \\ x_n\end{bmatrix} \end{equation*}
    is any column vector in \(\R^n\text{,}\) the matrix-vector product \(A\mathbf{x}\) is defined to be the column vector in \(\R^m\) defined by
    \begin{equation*} A\mathbf{x}=x_1\mathbf{a}_1+x_2\mathbf{a}_2+\cdots x_n\mathbf{a}_n\text{.} \end{equation*}

    Activity 2.2.3. Computing a Matrix-Vector Product.

    Compute the matrix-vector product \(A\mathbf{x}\) where
    \begin{equation*} A = \begin{bmatrix} -1 \amp 4 \amp -5 \\ 3 \amp 1 \amp -2 \end{bmatrix} \quad\text{and}\quad \mathbf{x} = \begin{bmatrix}2 \\ -3 \\ 4\end{bmatrix} \end{equation*}
    Solution.
    Write
    \begin{equation*} A = \begin{bmatrix}\mathbf{a}_1 \amp \mathbf{a}_2 \amp \mathbf{a}_3\end{bmatrix} \end{equation*}
    where
    \begin{equation*} \mathbf{a}_1 = \begin{bmatrix}-1 \\ 3\end{bmatrix} \quad \mathbf{a}_2 = \begin{bmatrix}4 \\ 1\end{bmatrix} \quad \mathbf{a}_3 = \begin{bmatrix}-5 \\ -2\end{bmatrix}\text{.} \end{equation*}
    Then
    \begin{align*} A\mathbf{x} \amp= 2\mathbf{a}_1 + (-3)\mathbf{a}_2 + 4\mathbf{a}_3\\ \amp= 2\begin{bmatrix}-1 \\ 3\end{bmatrix} + (-3)\begin{bmatrix}4 \\ 1\end{bmatrix} + 4\begin{bmatrix}-5 \\ -2\end{bmatrix}\\ \amp= \begin{bmatrix}-2 \\ 6\end{bmatrix} + \begin{bmatrix}-12 \\ -3\end{bmatrix} + \begin{bmatrix}-20 \\ -8\end{bmatrix}\\ \amp= \begin{bmatrix}-34 \\ -5\end{bmatrix} \end{align*}
    Notice that we can now write a linear system of equations as a single matrix equation: Consider a linear system of \(m\) equations in \(n\) unknowns of the form
    \begin{equation*} \begin{array}{cccccc} a_{11}x_1 \amp + \amp a_{12}x_2 \amp + \amp \cdots \amp + \amp a_{1n}x_n \amp = \amp b_1 \\ a_{21}x_1 \amp + \amp a_{21}x_2 \amp + \amp \cdots \amp + \amp a_{2n}x_n \amp = \amp b_2 \\ \vdots \amp \amp \vdots \amp \amp \amp \amp \vdots \amp \amp \vdots \\ a_{m1}x_1 \amp + \amp a_{m2}x_2 \amp + \amp \cdots \amp + \amp a_{mn}x_n \amp = \amp b_m. \end{array} \end{equation*}
    From this system, define the matrices
    \begin{equation*} A = \begin{bmatrix} a_{11} \amp a_{12} \amp \cdots \amp a_{1n} \\ a_{21} \amp a_{22} \amp \cdots \amp a_{2n} \\ \vdots \amp \vdots \amp \ddots \amp \vdots \\ a_{m1} \amp a_{m2} \amp \cdots \amp a_{mn} \end{bmatrix} \quad \mathbf{x} = \begin{bmatrix}x_1 \\ x_2 \\ \vdots \\ x_n\end{bmatrix} \quad \mathbf{b} = \begin{bmatrix}b_1 \\ b_2 \\ \vdots \\ b_m\end{bmatrix}\text{.} \end{equation*}

    Activity 2.2.4. Matrix Equation for Linear System.

    Using the matrices defined above, how can we rewrite the linear system as a single matrix equation?
    Solution.
    We can rewrite this system as the matrix equation: \(A\mathbf{x}=\mathbf{b}\text{.}\)
    1. Every system of linear equations has the form \(A\mathbf{x}=\mathbf{b}\) where \(A\) is the coefficient matrix, \(\mathbf{b}\) is the constant matrix/vector, and \(\mathbf{x}\) is the matrix/vector of variables.
    2. The system \(A\mathbf{x} =\mathbf{b}\) is consistent if and only if \(\mathbf{b}\) is a linear combination of the columns of \(A\text{.}\)

    Activity 2.2.5. Linear Combination of Columns.

    Is the vector
    \begin{equation*} \mathbf{b}=\begin{bmatrix} 1 \\ 2 \\ 4\end{bmatrix} \end{equation*}
    a linear combination of the vectors
    \begin{equation*} \mathbf{a}_1=\begin{bmatrix} 1 \\3 \\-2\end{bmatrix},\quad \mathbf{a}_2=\begin{bmatrix} -2 \\1 \\1\end{bmatrix},\quad\text{and}\quad \mathbf{a}_3=\begin{bmatrix} -3 \\5 \\0\end{bmatrix}\text{?} \end{equation*}
    Solution.
    We need to solve the system:
    \begin{equation*} x_1 \begin{bmatrix} 1 \\ 3 \\ -2 \end{bmatrix} + x_2 \begin{bmatrix} -2 \\ 1 \\ 1 \end{bmatrix} + x_3 \begin{bmatrix} -3 \\ 5 \\ 0 \end{bmatrix} = \begin{bmatrix} 1 \\ 2 \\ 4 \end{bmatrix}\text{,} \end{equation*}
    which corresponds to the augmented matrix:
    \begin{equation*} \left[ \begin{array}{rrr|r} 1 \amp -2 \amp -3 \amp 1 \\ 3 \amp 1 \amp 5 \amp 2 \\ -2 \amp 1 \amp 0 \amp 4 \end{array} \right]\text{.} \end{equation*}
    Start applying the Gaussian algorithm:
    \begin{gather*} R_2 \to R_2 - 3R_1\\ R_3 \to R_3 + 2R_1 \end{gather*}
    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp -2 \amp -3 \amp 1 \\ 0 \amp 7 \amp 14 \amp -1 \\ 0 \amp -3 \amp -6 \amp 6 \end{array}\right] \end{equation*}
    \begin{equation*} R_2 \to (1/7) R_2\text{:} \end{equation*}
    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp -2 \amp -3 \amp 1 \\ 0 \amp 1 \amp 2 \amp -\frac{1}{7} \\ 0 \amp -3 \amp -6 \amp 6 \end{array}\right] \end{equation*}
    \begin{equation*} R_3 \to R_3 + 3R_2\text{:} \end{equation*}
    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp -2 \amp -3 \amp 1 \\ 0 \amp 1 \amp 2 \amp -\frac{1}{7} \\ 0 \amp 0 \amp 0 \amp \frac{39}{7} \end{array}\right] \end{equation*}
    The last row of this matrix implies the system is inconsistent, so no solution exists. Thus, \(\mathbf{b}\) is not a linear combination of the given vectors.

    Activity 2.2.6. Special Matrix-Vector Products.

    (a)
    If \(\mathbf{x}=\mathbf{0}\text{,}\) what is \(A\mathbf{x}\text{?}\)
    Solution.
    If \(\mathbf{x}=\mathbf{0}\text{,}\) then \(A\mathbf{x}=\mathbf{0}\text{.}\)
    (b)
    If \(A\) is the zero matrix, what is \(A\mathbf{x}\text{?}\)
    Solution.
    If \(A\) is the zero matrix, then \(A\mathbf{x}=\mathbf{0}\)

    Subsection Matrix-Vector Product Properties

    Theorem 2.2.3.

    Let \(A\) and \(B\) be \(m\times n\) matrices, and let \(\mathbf{x}\) and \(\mathbf{y}\) be column vectors in \(\R^n\text{.}\) Then:
    • \(\displaystyle A(\mathbf{x}+\mathbf{y})=A\mathbf{x}+A\mathbf{y}\)
    • \(A(r\mathbf{x})=r(A\mathbf{x})=(rA)\mathbf{x}\) for all scalars \(r\)
    • \(\displaystyle (A+B)\mathbf{x}=A\mathbf{x}+B\mathbf{x}\)
    These are distributive properties of matrix-vector products.

    Activity 2.2.7. Proof of Scalar Distribution Property.

    Prove that
    \begin{equation*} A(r\mathbf{x})=r(A\mathbf{x})=(rA)\mathbf{x} \end{equation*}
    for any scalar \(r\) and matrix \(A\text{.}\)
    Solution.
    By the definition of matrix-vector multiplication, if \(A\) has columns \(\mathbf{a}_1, \mathbf{a}_2, \dots, \mathbf{a}_n\text{,}\) then:
    \begin{equation*} A\mathbf{x} = x_1 \mathbf{a}_1 + x_2 \mathbf{a}_2 + \dots + x_n \mathbf{a}_n \end{equation*}
    Multiplying \(\mathbf{x}\) by \(r\) gives:
    \begin{equation*} r\mathbf{x} = \begin{bmatrix} r x_1 \\ r x_2 \\ \vdots \\ r x_n \end{bmatrix} \end{equation*}
    Applying \(A\) to this vector:
    \begin{align*} A(r\mathbf{x}) \amp = (r x_1) \mathbf{a}_1 + (r x_2) \mathbf{a}_2 + \dots + (r x_n) \mathbf{a}_n\\ \amp = r (x_1 \mathbf{a}_1 + x_2 \mathbf{a}_2 + \dots + x_n \mathbf{a}_n) = r(A\mathbf{x}) \end{align*}
    If we multiply each column \(\mathbf{a}_j\) of \(A\) by \(r\text{,}\) we find that:
    \begin{equation*} rA = \begin{bmatrix} r\mathbf{a}_1 \amp r\mathbf{a}_2 \amp \dots \amp r\mathbf{a}_n \end{bmatrix} \end{equation*}
    Applying \(rA\) to \(\mathbf{x}\) gives:
    \begin{align*} (rA)\mathbf{x} \amp = x_1 (r\mathbf{a}_1) + x_2 (r\mathbf{a}_2) + \dots + x_n (r\mathbf{a}_n)\\ \amp = r(x_1 \mathbf{a}_1 + x_2 \mathbf{a}_2 + \dots + x_n \mathbf{a}_n)\\ \amp = A(r\mathbf{x})\text{.} \end{align*}

    Subsection Matrix-Vector Products Using Dot Products

    We can express Matrix-Vector multiplication using dot products. Recall the definition of the dot product of two vectors from Definition 1.1.12. Since row and column vectors can be considered as vectors, we can consider their dot products.

    Activity 2.2.8. Computing a Dot Product.

    Compute the dot product of \((-2, \pi, 0, 3/4)\) and \((3, 7, -4, 12)\text{.}\)
    Solution.
    \begin{align*} (-2)(3) + (\pi)(7) + (0)(-4) + (3/4)(12) \amp = -6 + 7\pi + 0 + 9\\ \amp = 7\pi + 3 \end{align*}

    Theorem 2.2.4.

    Let \(A\) be an \(m\times n\) matrix and let \(\mathbf{x}\) be a column vector in \(\R^n\text{.}\) Then each entry of the vector \(A\mathbf{x}\) is the dot product of the corresponding row of \(A\) with \(\mathbf{x}\) (see Figure 2.2.5).

    Activity 2.2.9. Matrix-Vector Product via Dot Products.

    Compute the product \(A\mathbf{x}\) using the dot product, where
    \begin{equation*} A = \begin{bmatrix} -1 \amp 4 \amp -5 \\ 3 \amp 1 \amp -2 \end{bmatrix} \quad\text{and}\quad \mathbf{x} = \begin{bmatrix}2 \\ -3 \\ 4\end{bmatrix}\text{.} \end{equation*}
    Solution.
    \begin{align*} A\mathbf{x} \amp= \begin{bmatrix} -1 \amp 4 \amp -5 \\ 3 \amp 1 \amp -2 \end{bmatrix} \begin{bmatrix} 2 \\ -3 \\ 4 \end{bmatrix}\\ \amp = \begin{bmatrix} (-1,4,-5) \cdot (2,-3,4) \\ (3,1,-2) \cdot (2,-3,4) \end{bmatrix}\\ \amp= \begin{bmatrix} (-1)(2) + (4)(-3) + (-5)(4) \\ (3)(2) + (1)(-3) + (-2)(4) \end{bmatrix}\\ \amp= \begin{bmatrix} -34 \\ -5 \end{bmatrix} \end{align*}

    Remark 2.2.6.

    Conversely, the dot product can be calculated in terms of matrix multiplication. If we consider two \(n\)-vectors \(\mathbf{v}\) and \(\mathbf{w}\text{,}\) then the scalar \(\mathbf{v} \cdot \mathbf{w}\) is equal to the only entry in the \(1 \times 1\) matrix \(\mathbf{v}^T \mathbf{w}\text{.}\)

    Definition 2.2.7. The Identity Matrix.

    For each \(n \geq 2\text{,}\) the identity matrix \(I_n\) is the \(n\times n\) matrix with 1s on the main diagonal (upper left to lower right), and zeros elsewhere.
    For example, we have
    \begin{equation*} I_2 = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix}, \quad I_3 = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}, \quad\text{and}\quad I_4 = \begin{bmatrix} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \end{bmatrix}\text{.} \end{equation*}

    Fact 2.2.8.

    For each \(n\geq 2\) and each column vector \(\mathbf{x} \in \R^n\text{,}\) \(I_n\mathbf{x}=\mathbf{x}\text{.}\)

    Activity 2.2.10. Verifying Identity Matrix Property.

    Verify that \(I_3 \mathbf{x} = \mathbf{x}\) for all column vectors \(\mathbf{x} \in \R^3\text{.}\)
    Solution.
    \begin{align*} I_3 \mathbf{x} \amp= \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \\ x_3 \end{bmatrix}\\ \amp= \begin{bmatrix} x_1 \\ x_2 \\ x_3 \end{bmatrix} = \mathbf{x} \end{align*}

    Subsection Standard Basis Vectors

    Definition 2.2.9. The Standard Basis.

    Let \(\mathbf{e}_j\) be the \(j\)th column of \(I_n\text{,}\) i.e.,
    \begin{equation*} \mathbf{e}_j = \begin{bmatrix} t_1 \\ t_2 \\ \vdots \\ t_n \end{bmatrix} \end{equation*}
    where \(t_j = 1\text{,}\) but \(t_i = 0\) for all \(i \neq j\text{.}\) These vectors are called the standard basis vectors for \(\R^n\) (the word ‘basis’ will be defined in Definition 4.2.10).
    In \(\R^2\text{,}\) it is sometimes common to use the notation \(\mathbf{i}\) and \(\mathbf{j}\) for the standard basis \(\mathbf{e}_1\) and \(\mathbf{e}_2\text{.}\) Similarily, in \(\R^3\) it is common to use the notation \(\mathbf{i}\text{,}\) \(\mathbf{j}\text{,}\) and \(\mathbf{k}\) for the standard basis \(\mathbf{e}_1\text{,}\) \(\mathbf{e}_2\text{,}\) and \(\mathbf{e}_3\text{.}\) This notation is most common amongst physicists and engineers.

    Activity 2.2.11. Finding a Standard Basis Vector.

    For \(\R^4\text{,}\) what is \(\mathbf{e}_3\text{?}\)
    Solution.
    \begin{equation*} \mathbf{e}_3 = \begin{bmatrix} 0 \\ 0 \\ 1 \\ 0 \end{bmatrix} \end{equation*}

    Activity 2.2.12. Matrix-Vector Product with Standard Basis Vector.

    Compute \(A\mathbf{e}_3\) for
    \begin{equation*} A=\begin{bmatrix} 2 \amp 0 \amp 1 \amp -1 \\ 3 \amp -1 \amp 4 \amp 0 \\ 2 \amp -2 \amp 3 \amp -1 \end{bmatrix} \end{equation*}
    Solution.
    \begin{equation*} A\mathbf{e}_3 = \begin{bmatrix} 2 \amp 0 \amp 1 \amp -1 \\ 3 \amp -1 \amp 4 \amp 0 \\ 2 \amp -2 \amp 3 \amp -1 \end{bmatrix} \begin{bmatrix} 0 \\ 0 \\ 1 \\ 0 \end{bmatrix} = \begin{bmatrix} 1 \\ 4 \\ 3 \end{bmatrix} \end{equation*}

    Fact 2.2.10.

    If \(A=\begin{bmatrix}\mathbf{a}_1 \amp \mathbf{a}_2 \amp\cdots \amp \mathbf{a}_n\end{bmatrix}\) is an \(m\times n\) matrix, then for \(j = 1 \ldots n\text{,}\)
    \begin{equation*} A\mathbf{e}_j=\mathbf{a}_j\text{.} \end{equation*}

    Theorem 2.2.11.

    Let \(A\) and \(B\) be \(m\times n\) matrices. If \(A\mathbf{x}=B\mathbf{x}\) for all \(\mathbf{x}\) in \(\R^n\text{,}\) then \(A=B\text{.}\)

    Activity 2.2.13. Proof of Matrix Equality.

    Prove the theorem above.
    Solution.
    Since \(A\mathbf{x} = B\mathbf{x}\) for all \(\mathbf{x}\text{,}\) choose \(\mathbf{x} = \mathbf{e}_j\) for \(j=1,\dots,n\text{.}\) Then \(A\mathbf{e}_j = B\mathbf{e}_j\) for all \(j\text{.}\) But using Fact 2.2.10, this means that the \(j\)th column of \(A\) is equal to the \(j\) column of \(B\text{.}\) Since this is true for all \(j\text{,}\) \(A=B\text{.}\)

    Subsection Transformations

    Geometric operations can be described via matrix-vector products.

    Definition 2.2.12. Transformations.

    A function \(T: \R^n \to \R^m\) is called a transformation from \(\R^n\) to \(\R^m\text{.}\) A transformation \(T\) assigns to each vector \(\mathbf{x}\) in \(\R^n\) a uniquely determined vector \(T(\mathbf{x})\) in \(\R^m\text{,}\) called the image of \(\mathbf{x}\) under \(T\text{.}\)

    Activity 2.2.14. Finding Image of a Vector.

    Consider \(T:\R^3\to\R^2\) given by
    \begin{equation*} T( (x_1,x_2,x_3) ) = (x_1 + x_3, x_2)\text{.} \end{equation*}
    What is the image of the vector \((3,-2,4)\) under \(T\text{?}\)
    Solution.
    \begin{equation*} T((3,-2,4)) = ( 3 + 4, -2 ) = (7,-2)\text{.} \end{equation*}

    Definition 2.2.13. Matrix Transformations.

    Given an \(m\times n\) matrix \(A\text{,}\) we can consider the transformation \(T_A: \R^n \to \R^m\) given by multiplying vectors by \(A\) (viewing those vectors as column vectors), i.e.
    \begin{equation*} T_A(\mathbf{x}) = A \mathbf{x}\text{.} \end{equation*}
    For each \(\mathbf{x}\) in \(\R^n\text{.}\) \(T_A\) is called the matrix transformation induced by \(A\text{.}\)

    Activity 2.2.15. Reflection Transformation.

    Consider the transformation in \(\R^2\) given by reflection across the \(y\)-axis, i.e. so that the vector \((a_1,a_2)\) is sent to the vector \((-a_1,a_2)\text{.}\) We can represent this reflection via a matrix-vector product. Find the matrix.
    Solution.
    \begin{equation*} A = \begin{bmatrix} -1 \amp 0 \\ 0 \amp 1 \end{bmatrix} \end{equation*}

    Activity 2.2.16. Identifying Matrix Transformations.

    Is the transformation defined by
    \begin{equation*} T(x_1,x_2) = (x_1 + x_2, x_2) \end{equation*}
    a matrix transformation?
    Solution.
    Yes, \(T\) is a matrix transformation, since it can be written as \(T_A\) where
    \begin{equation*} A = \begin{bmatrix} 1 \amp 1 \\ 0 \amp 1 \end{bmatrix} \end{equation*}
