Theorem 4.2.2.

The following properties of the orthogonal complement hold:
  1. For any subspace \(U\) of \(\R^n\text{,}\) \(U^\perp\) is a subspace of \(\R^n\)
    🔗
  2. \(\{0\}^\perp = \R^n\) and \((\R^n)^\perp = \{0\}\)
    🔗
  3. If \(U = \spans \{\mathbf{x}_1,\ldots,\mathbf{x}_k\}\text{,}\) then
    \begin{equation*} U^\perp = \{\mathbf{x} \text{ in } \R^n \mid \mathbf{x} \cdot \mathbf{x}_i = 0 \text{ for } i=1,2,\ldots,k\} \end{equation*}
    🔗
